# Introduction 
Api em .Net 5 com autenticação JWT e SWAGGER e projeto de teste.

# Getting Started
1.	Restaure os pacotes nuget necessários 
2. Configura o tempo de expiração do token no arquivo appsettings.json
3.	Execute o projeto
4.	Se preferir pode utilizar o arquivo JSON disponível na pasta "RequestHelper" dentro do projeto (utilize Postman, Insomnia ou outro software de sua preferência.)
